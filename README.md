# CSV Analyzer

A lightweight web application for quick data exploration and statistical analysis of CSV files.

## Features

- 📤 **Upload CSV files** directly through the browser
- 📊 **View data** as an interactive table
- 📈 **Basic statistics**: min, max, mean, quartiles for numeric columns
- 📉 **Visualizations**: histograms for selected numeric columns

## Tech Stack

- Python 3.14
- Streamlit
- Pandas
- Matplotlib

## Installation

```bash
git clone https://github.com/irina-kirillova/csv-analyzer.git
cd csv-analyzer
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

Open your browser and navigate to `http://localhost:8501`

1. Upload a CSV file
2. View the data table
3. Explore statistics and visualizations
